## Installation

:bulb: Clone this repo 
:bulb: add .env.local
:bulb: Then in project folder

Run

```sh
$ composer install
$ bin/console d:d:c
$ bin/console make:migration 
$ bin/console doctrine:migration:migrate 
$ bin/console doctrine:fixtures:load 
$ symfony server:run 
```
